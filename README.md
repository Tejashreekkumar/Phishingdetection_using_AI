# PhishGuard AI - Phishing URL Detection System

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run: `python app.py`
3. Open: http://localhost:5000

## Default Login
- Username: `admin`
- Password: `admin`

## Features
- AI-powered phishing URL detection with 6 models
- 30+ URL feature extraction & analysis
- Explainable AI (reasons for detection)
- Model comparative analysis with ensemble voting
- User authentication (login, register, forgot password)
- Detection history with database storage
- Cybercrime complaint reporting
- Admin dashboard with statistics

## Models
- Naive Bayes (TF-IDF) - ML model
- Random Forest (URL Features)
- SVM (Security Features)
- Decision Tree (Domain Features)
- Gradient Boosting (Ensemble)
- Feature Analysis (Rule-based)
